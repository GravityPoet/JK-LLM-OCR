# JK-LLM-OCR

最强本地OCR插件。

## 安装
1. 下载仓库中的 `JK-LLM-OCR.bobplugin` 或从 Release 下载。  
2. 双击安装到 Bob。

## 更新通道
- 插件更新元信息在仓库根目录 `appcast.json`。
- Bob 会按 `info.json` 中的 `appcast` 地址检查更新。

## 手动启动本地 OCR 服务（示例）
```bash
cd /Users/moonlitpoet/Tools/PaddleOCR/bob-plugin-ppocrv5-server
./scripts/start_ppocrv5_server.sh
```
