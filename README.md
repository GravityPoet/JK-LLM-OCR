# JK-LLM-OCR

最强本地OCR插件。

## 插件文件
- `JK-LLM-OCR.bobplugin`

## 使用方式
1. 双击或使用 `open JK-LLM-OCR.bobplugin` 安装到 Bob。
2. 启动本地 OCR 服务后，在 Bob 中调用 OCR 即可。

## 本地服务手动启动（示例）
```bash
cd /Users/moonlitpoet/Tools/PaddleOCR/bob-plugin-ppocrv5-server
./scripts/start_ppocrv5_server.sh
```
