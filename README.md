# JK-LLM-OCR

最强本地OCR插件。

## 插件文件
- `JK-LLM-OCR.bobplugin`

## 安装
1. 在仓库中下载 `JK-LLM-OCR.bobplugin`
2. 双击或执行 `open JK-LLM-OCR.bobplugin` 安装到 Bob

## 更新检查
- 本仓库根目录提供 `appcast.json`，用于 Bob 检查插件更新。

## 本地 OCR 服务手动启动
```bash
cd /Users/moonlitpoet/Tools/PaddleOCR/bob-plugin-ppocrv5-server
./scripts/start_ppocrv5_server.sh
```
